# Quick Start Guide

## üöÄ Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Open browser**:
   Navigate to `http://localhost:3000`

4. **Create an account**:
   - Click "Sign Up" on the login page
   - Enter your name, email, and password
   - You'll be automatically logged in

5. **Start using AR CPR Guidance**:
   - Select "CPR" from the home screen
   - Click "Start AR Guidance"
   - Grant camera permissions when prompted
   - Follow the on-screen instructions and voice guidance

## üì± Testing on Mobile

For best results, test on a mobile device:

1. **Local Network Access**:
   - Find your computer's IP address (e.g., `192.168.1.100`)
   - On your mobile device, navigate to `http://YOUR_IP:3000`

2. **HTTPS Required for Production**:
   - WebAR requires HTTPS in production
   - Use services like Vercel, Netlify, or ngrok for HTTPS testing
